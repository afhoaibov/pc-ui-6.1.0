地区选择器

> 选择地区，返回选中的平铺一维数组，和选中的集合树。


#### 支持传参

| 参数         | 说明                                                     | 类型     | 可选值     | 默认值                                         | 必传 |
| :----------- | -------------------------------------------------------- | -------- | ---------- | ---------------------------------------------- | ---- |
| show         | 是否显示选择器，支持 .sync 修饰符。                      | Boolean  | true/false | false                                          | 是   |
| api          | 获取地区数据API。                                        | String   |            | -                                              | 是   |
| default-ids  | 默认数据的id集合，一般是地区的code集合。                 | Array    |            | -                                              | 否   |
| disabled-ids | 需要禁用的id集合，一般是地区的code集合。                 | Array    |            | -                                              | 否   |
| exclude-ids  | 需要排除的id集合，一般是地区的code集合。                 | Array    |            | -                                              | 否   |
| props        | 配置项，一般是数据的字段名称。例如{id: 'region_code'}    | Object   |            | {id: 'id', name: 'name', children: 'children'} | 否   |
| confirm      | 确定选择方法，支持异步。如果返回reject，不会关闭选择器。 | Function |            |                                                | 否   |

#### 触发事件

| 名称    | 说明               | 回调参数                                                     |
| ------- | ------------------ | ------------------------------------------------------------ |
| closed  | 选择器关闭事件     |                                                              |
| confirm | 选择器确认方法调用 | (Items: Array, itemsTree: Array)。items：选中的集合一维数组，ItemsTree：选中的集合树。 |
