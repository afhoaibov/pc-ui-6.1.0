素材选择器

> 素材选择器可选择图片和视频
>
> 同时也可以简单上传图片和视频


#### 支持传参

| 参数       | 说明                                                     | 类型     | 可选值           | 默认值 | 必传 |
| :--------- | -------------------------------------------------------- | -------- | ---------------- | ------ | ---- |
| show       | 是否显示选择器，支持 .sync 修饰符。                      | Boolean  |                  | false  | 是   |
| upload-api | 文件上传接口，必传。                                     | String   |                  | flase  | 是   |
| type       | 素材类型，如果不传的话，图片、视频都可以选择。           | String   | 不传/image/video |        | 否   |
| request    | 发起请求的方法，一般是utils里的request.js。              | Function |                  |        | 是   |
| confirm    | 确定选择方法，支持异步。如果返回reject，不会关闭选择器。 | Function |                  |        | 否   |
| limit      | 最大选择个数，-1表示无限制。                             | Number   | -1~正无穷        | -1     | 否   |

#### 触发事件

| 名称    | 说明               | 回调参数                                       |
| ------- | ------------------ | ---------------------------------------------- |
| closed  | 选择器关闭事件     |                                                |
| confirm | 选择器确认方法调用 | (Items: Array)如果没有选择项目，回调是空数组。 |
